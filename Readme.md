## :dart: About

This is a simple image gallery project developed using HTML, CSS and JavaScript

## :sparkles: Features

:heavy_check_mark: Feature 1 - View Images in gallery\
:heavy_check_mark: Feature 2 - Pagination for gallery where only 6 images will show in one page\
:heavy_check_mark: Feature 3 - Filter images using categories\
:heavy_check_mark: Feature 4 - View image in large scale and navigate through images

## :rocket: Technologies

The following tools were used in this project:

- HTML
- CSS
- JavaScript

## :checkered_flag: Starting

```bash
# Clone this project
$ git clone https://github.com/DeepikaSharma5/Image-Gallery.git

# Run the project
$ Double click index.html

```
